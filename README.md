# Analog Clock

This project demonstrates an **analog clock** built using **HTML**, **CSS**, and **JavaScript**. The clock accurately displays the current time with three hands: **hour**, **minute**, and **second**.

## Learning Objectives

- ✅ Create an analog clock with three hands (hour, minute, second).
- ✅ Ensure the second hand updates its position based on the current time.
- ✅ Maintain a responsive design that works well on all devices.
- ✅ Keep the HTML file unchanged.
- ✅ Write well-documented CSS and JavaScript code.
- ✅ Format the code according to [W3Schools JavaScript conventions](https://www.w3schools.com/js/js_conventions.asp).
- ✅ Use English and follow international coding standards.
- ✅ Publish the project on GitHub as a public repository.
- ✅ Include a `github-repo.txt` file containing the repository link in the submission ZIP.
- ✅ Provide a `README.md` file to describe the project.
